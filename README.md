🎭 Playwright Automation - Practical 01 🚀
An end-to-end automation suite targeting rahulshettyacademy.com, designed to master modern web testing techniques.

📌 Project Overview
This project is a collection of Playwright test scripts focused on the Rahul Shetty Academy Practice Page. It serves as a practical implementation of automation concepts like handling dynamic content, frames, alerts, and synchronization.

✨ Key Scenarios Covered
🖱️ UI Controls: Checkboxes, Radio Buttons, and Static/Dynamic Dropdowns.

🔀 Switching Windows: Handling child tabs and pop-up windows.

🖼️ iFrames: Automating interactions inside embedded frames.

⚠️ Alerts & Dialogs: Accepting/Dismissing JavaScript alerts and confirms.

📊 Web Tables: Extracting and validating data from dynamic tables.

🕵️ Visibility: Handling hidden/visible elements and hover actions.

🛠️ Tech Stack
Framework: Playwright

Language:JavaScript

Test Runner: Playwright Built-in Runner

Reporting: Playwright HTML Report
